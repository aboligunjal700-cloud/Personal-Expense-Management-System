# Personal-Expense-Management-System
A simple python project based on Personal Expense Management System. it can help to record and manage their expenses so, they have a rough idea about how much they spend already.we can add expense with date,category,and amount. this makes money management easy.

#Project Overview:<br> 
In today's busy lifestyle,people often lose track of how and where their money is being spent . so, I have created a project and my main aim is to solve that issue by creating a expense tracker to record expenses with date ,category (like:Food,Shopping,Travelling),and the amount you spent and one special feature I added that Firstly you have to set a spending limit and when you enter your value if this value is greater than your spending limit you will get a warning that you have crossed your spending limit and be carefull.you can calculate the total spent amount as well .so because of this user can find where the money actually goes.so yaa it will turned out to be beneficial for You.

#Features:<br> 
Set a Spending limit <br>  Add expense <br> view total expense with dates,categories <br>  Shows remaining amount after showing Total Expense <br> Gives limit alert when expenses exceed the set limit
